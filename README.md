ЗАДАНИЕ 3

1)наименование: bid
2)
a)Server-Side Authorization Check
-Обязательная проверка прав доступа на сервере для каждого запроса к корзине
-Проверять, принадлежит ли запрашиваемая корзина текущему авторизованному пользователю
-Не доверять клиентским данным при проверке доступа

b)Использование непредсказуемых идентификаторов
-Замена последовательных ID на UUID или криптографически стойкие случайные значения
-Использование длинных случайных строк вместо простых чисел (1, 2, 3...)

c)Session-Based Access Control
-Привязка корзины к сессии пользователя, а не к ID в URL
-Автоматическое определение корзины пользователя из его сессии/токена

d)Ролевая модель доступа
-Реализация ролевой модели (RBAC) с четкими правами доступа
-Проверка не только принадлежности корзины, но и прав пользователя на операцию
